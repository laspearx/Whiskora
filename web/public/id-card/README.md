# Whiskora Pet ID Card Assets

Use URL-safe English file names for Pet ID card assets.

Pet ID card backgrounds should be visual-only. Do not embed readable text, Whiskora logos, QR codes, pet data, CTA buttons, or verification seals into the background image. The card page should render those elements separately so they stay sharp, searchable, translatable, and responsive.
